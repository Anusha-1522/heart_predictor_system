# ❤️ HeartCare AI - Heart Disease Prediction System

HeartCare AI is a web-based Heart Disease Prediction and Cardiac Patient Management System developed using Flask, HTML, CSS, and JavaScript. The application helps healthcare professionals predict heart disease risk, manage patient records, and monitor patient health through an interactive dashboard.
---

## ✨ Features

- Secure Login Authentication
- Heart Disease Risk Prediction
- Patient Management (Add, View, Delete)
- Interactive Dashboard
- Real-time Risk Analysis
- Patient Statistics and Analytics
- Responsive User Interface
---

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- Flask
- Flask-CORS

### Database
- JSON (patients.json)
